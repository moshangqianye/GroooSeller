# grooo_seller_Android
====================

咕噜卖家Android版，为咕噜校园的卖家制作。

 - [咕噜校园官网](http://www.grooo.cn/)
 - [咕噜校园代码](https://github.com/RunziiMo/grooo_android_client)

## 项目介绍

 1. 采用Material Design
 2. 采用极光推送sdk
 3. 主要第三方库
 -  material-dialogs
 - glide
 - retrofit
 - rxjava
 - butterknife

 4. 主要设计结构
  - MVVM(databinding)
  - RX
